### Ollama local
* Download from Ollama site - https://ollama.com/download
* Pull a model locally. Stored locally at <user_home>\.ollama
```
ollama pull qwen2:7b
```
* Start Ollama in the background
```
ollama serve
```
* Other commands
```
ollama list
ollama show <model_name>
```
